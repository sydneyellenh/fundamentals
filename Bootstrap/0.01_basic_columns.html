Bootstrap Notes

Bootstrap is made up of a 12-column grid system. You add your content into the 'rows' and 'columns' using Bootstrap CSS classes. Bootstrap will then take care of the responsiveness work for us.

Containers, Rows and Columns

Because Bootstrap takes care of the resposiveness for us, we have to format things in a particular way:

  1. Containers surround the whole section of rows and columns
  2. Rows represent a horizontal line of elements
  3. Columns represent the horizontal space that elements take up in a row

Types of Containers:

  • Fixed container: 'container'
    • Has a fixed width, which changes at certain breakpoints
    • Has built-in padding of 15px all the way around
  • Fluid container: 'container-fluid'
    • Has a fluid width, which stretches to 100% of its parent element
    • Has built-in padding of 15px all the way around

**Once you get down to the mobile size screens, they look pretty much identical, it's at medium and large screens where you can see the differences**

Columns

  • Column width can be any integer between 1 and 12
  • We assign elements within a row, a column width
  • We do this by using Bootstrap classes:
    • Example format: col-{size}-{number}
    • {size} = the size of the viewport (xs,sm,md,lg)
    • {number} = number of columns you want the element to span
    • Example: "col-xs-12" or "col-md-6"

Viewport sizes

  • xs( < 768px)
  • sm( >= 768px)
  • md( >= 992px)
  • lg( >= 1200px)
